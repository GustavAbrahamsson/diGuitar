# SDP810

![SDP810](https://pschatzmann.github.io/arduino-audio-tools/doc/resources/ut2uh_sdp810_mic.jpeg)

The SDP810 is a 16 bit differential pressure sensor connected via I2C.

### Pins:
 
| SDP810  | ESP32
|---------|---------------
| SCL     | SCL GPIO22
| VDD     | 3.3
| GND     | GND
| SDA     | SDA GPIO21
